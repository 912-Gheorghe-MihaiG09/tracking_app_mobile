# Mobile Tracking App

## Application Details

This Flutter application was designed for IOS and Android devices and is intended to be used for tracking IOT devices

This Application provides the following features:

1. pairing device using serial numbers - by scanning a qr code with the phones camera or by adding the serial number manually
2. managing devices - delete & rename
3. seeing live location of devices
4. pinging devices make them easier to be found - pinging a device will cause the device to play a song
5. assigning a geofence for a device - if the device leaves the giofence a notification will be received
